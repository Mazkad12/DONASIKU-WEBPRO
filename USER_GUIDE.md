## <a name="pendahuluan"></a>1. Pendahuluan

**Donasiku** adalah jembatan kebaikan yang menghubungkan orang yang memiliki barang layak pakai dengan mereka yang membutuhkan. Platform ini gratis dan terbuka untuk umum.

---

## <a name="memulai-aplikasi"></a>2. Memulai Aplikasi

### <a name="registrasi-akun"></a>Registrasi Akun

Sebelum dapat bertransaksi, Anda wajib memiliki akun.
1.  Buka halaman utama aplikasi.
2.  Klik tombol **"Daftar Sekarang"** atau link "Belum punya akun?".
3.  Isi formulir pendaftaran:
    *   **Nama Lengkap**: Masukkan nama asli Anda.
    *   **Email**: Gunakan email aktif (verifikasi mungkin diperlukan).
    *   **Password**: Minimal 8 karakter, kombinasi huruf dan angka disarankan.
    *   **Peran (Role)**:
        *   Pilih **Donatur** jika Anda ingin menyumbang barang.
        *   Pilih **Penerima** jika Anda membutuhkan bantuan barang.
4.  Klik tombol **"Register"**.
5.  Jika sukses, Anda akan diarahkan ke halaman Login.

### <a name="login"></a>Login

1.  Masukkan **Email** yang terdaftar.
2.  Masukkan **Password**.
3.  Klik tombol **"Masuk"**.
4.  Anda akan diarahkan ke Dashboard sesuai peran Anda.
    *   *Catatan: Jika lupa password, silakan hubungi admin (fitur reset mandiri sedang dikembangkan).*

## <a name="panduan-untuk-donatur"></a>3. Panduan untuk Donatur

Peran ini dikhususkan bagi Anda yang ingin mendonasikan barang.

### <a name="dashboard-donatur"></a>Dashboard Donatur
Di halaman utama ini, Anda dapat melihat ringkasan aktivitas donasi Anda.
*   **Total Donasi**: Jumlah barang yang pernah Anda posting.
*   **Donasi Tersalurkan**: Barang yang sudah sampai ke tangan penerima.
*   **Donasi Aktif**: Barang yang masih menunggu permintaan.

### <a name="membuat-donasi-baru"></a>Membuat Donasi Baru
1.  Klik tombol **"+ Buat Donasi"** di pojok kanan atas atau menu navigasi.
2.  Unggah **Foto Barang**:
    *   Pastikan foto jelas dan terang.
    *   Format: JPG/PNG, ukuran maksimal 2MB.
3.  Isi **Nama Barang** (Contoh: "Kemeja Kantor Pria Size L").
4.  Pilih **Kategori** (Pakaian, Buku, Elektronik, dll).
5.  Tulis **Deskripsi**: Jelaskan kondisi barang, cacat (jika ada), dan alasan donasi.
6.  Tentukan **Lokasi**: Lokasi penjemputan barang.
7.  Klik **"Simpan Donasi"**.
8.  Barang Anda kini statusnya **Status: Tersedia** dan muncul di pencarian Penerima.

### <a name="mengelola-status-donasi"></a>Mengelola Status Donasi
Ketika ada Penerima yang mengajukan permintaan:
1.  Anda akan menerima notifikasi.
2.  Buka menu **"Permintaan Masuk"**.
3.  Anda bisa melihat profil calon penerima dan alasan mereka.
4.  Aksi:
    *   **Terima**: Menyetujui permintaan. Status barang berubah menjadi **Diproses**.
    *   **Tolak**: Menolak permintaan. Barang tetap **Tersedia** untuk orang lain.
5.  Setelah barang diserahkan/dikirim, klik tombol **"Konfirmasi Pengiriman"**. Status berubah menjadi **Dikirim**.

### <a name="melihat-riwayat-donatur"></a>Melihat Riwayat
Menu ini menampilkan arsip semua donasi yang telah selesai (Diterima oleh Penerima). Anda bisa melihat jejak kebaikan Anda di sini.

## <a name="panduan-untuk-penerima"></a>4. Panduan untuk Penerima

Peran ini bagi individu atau organisasi yang membutuhkan barang donasi.

### <a name="dashboard-penerima"></a>Dashboard Penerima
Halaman ini adalah etalase barang-barang yang tersedia. Anda akan melihat daftar barang terbaru yang diposting oleh Donatur.

### <a name="mencari-barang"></a>Mencari Barang
1.  Gunakan **Kolom Pencarian** di bagian atas untuk mengetik nama barang (misal: "Sepatu Sekolah").
2.  Gunakan **Filter Kategori** untuk menyaring hasil (misal: hanya tampilkan "Buku").
3.  Klik pada kartu barang untuk melihat **Detail Donasi** (Foto besar, Deskripsi lengkap, Lokasi).

### <a name="mengajukan-permintaan"></a>Mengajukan Permintaan
JIKA barang statusnya **Tersedia**:
1.  Klik tombol **"Ajukan Permintaan"** di halaman detail barang.
2.  Isi formulir singkat **Alasan Permintaan**:
    *   Ceritakan kenapa Anda membutuhkannya. Kejujuran meningkatkan peluang disetujui Donatur.
3.  Klik **"Kirim Permintaan"**.
4.  Status barang bagi Anda akan menjadi **Menunggu Persetujuan**.

## <a name="manajemen-akun"></a>5. Manajemen Akun

### <a name="edit-profil"></a>Edit Profil
1.  Klik foto profil Anda di pojok kanan atas -> **Profil Saya**.
2.  Klik tombol **"Edit Profil"**.
3.  Ubah Nama, No. Telepon, atau Alamat.
4.  Klik **"Simpan Perubahan"**.

### <a name="ganti-foto-profil"></a>Ganti Foto Profil
Di halaman Edit Profil:
1.  Klik ikon kamera pada avatar.
2.  Pilih foto dari galeri HP/Komputer Anda.
3.  Sistem akan otomatis mengunggah dan menampilkan pratinjau.

### <a name="logout"></a>Logout
Untuk keamanan, jangan lupa keluar jika menggunakan perangkat umum.
1.  Klik ikon profil.
2.  Pilih **"Keluar"**.
